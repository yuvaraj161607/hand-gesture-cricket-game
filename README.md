# Hand Gesture Cricket Game

A computer vision based cricket game where players score runs using hand gestures detected through a webcam.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## Features
- Hand gesture based run detection
- Player vs Computer mode
- Two Player mode
- Toss system
- Real-time score calculation

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the game

python main.py
